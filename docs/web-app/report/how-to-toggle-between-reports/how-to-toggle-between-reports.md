# How to toggle between reports

- **Navigate****to****the****Reports Management****Page:**On the Left-Hand Side (LHS)navigation menu, click on "**Reports**".

- **Select****a****Report****Type:**In the ReportsManagement page, at the top left corner of the screen, click on the dropdown menu under the "Reports" heading. From the dropdown menu, select the desired report type. (See figure below)

- The page will update to display the selected report type, allowing you to view the corresponding data.
