# Cut Sheet

The **Cut Sheet** dashboard is a quick view of **what needs cutting** — the order and square-metre totals for a date range, and how much has been harvested versus what's still to go.

## Where to find it

Top navigation → **Cut Sheet**.

## What it shows

Pick a **date range**, then see:

- **Total** — orders and square metres booked for the period.
- **Completed** — how much has been cut/completed.
- **Meters Harvested** — square metres already harvested.
- **Meters RTD** — square metres ready to deliver.
