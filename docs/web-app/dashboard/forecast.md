# Forecast

The **Forecast** dashboard is a **short-term look ahead** — order, delivery and laying volumes over the coming days — plus a list of **outstanding payments**.

## Where to find it

Top navigation → **Forecast**.

## What it shows

- **Order Forecast** — total square metres per day.
- **Delivery Forecast** — number of orders and pallets per day.
- **Laying Forecast** — number of orders and total square metres per day.
- **Outstanding Payments** — orders with amounts still owed (order #, customer, delivery date, due date, amount, status).
