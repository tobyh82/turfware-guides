# Forecast

The Payment and Forecast Overview component allows users to visualise order forecasts, delivery schedules, laying forecasts, and track outstanding payments. The component organises information into three main sections: Order Forecast, Delivery Forecast, and Laying Forecast, along with a table for managing outstanding payments. Users can view forecasts for the upcoming week (starting from today).

### Order Forecast

The Order Forecast section provides a daily overview of forecasted grass deliveries, grouped by grass type.

### Delivery Forecast

The Delivery Forecast section focuses on orders marked for delivery for the upcoming week.

### Laying Forecast

The Laying Forecast section focuses on orders that have installation items for the upcoming week.

### Outstanding Payments

- The Outstanding Payments section provides a table for managing invoices and tracking overdue payments.

- Features:

- Order Details: Displays the order number, customer name, delivery date, due date, invoice amount, and payment status.

- Export Functionality: Allows exporting the list of outstanding orders as a CSV file. By clicking the green button with the export icon. (See figure below).
