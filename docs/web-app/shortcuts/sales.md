# Sales

The Sales Overview component provides users with a comprehensive view of sales data filtered by date range and customer segments. It includes sections to analyse sales summaries, financial year comparisons, orders, and top customers. The component is designed to be interactive and customisable, allowing users to filter data based on their needs.

##

**Total****Sales and****Orders:**This section summarises the total sales and number of orders within the selected date range. (See Figure F below)

- Total Sales: Displays the total sales amount. It is calculated by summing the price of all orders within the date range.

- Total Orders: Shows the total number of orders within the specified period.

- Existing Customers: Displays the number of existing customers.

- New customer: Displays the number of new customers.
