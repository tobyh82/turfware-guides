# How to use filters

- **Identify****Available****Filters:**Filters are often available to narrow down the list based on specific criteria, such as order status. (See figure below)

- **Apply****Filters:** Click on the filter option, usually represented by dropdowns or checkboxes, to choose the criteria you wish to filter by.

- **View****FilteredResults:**The list will update to show only those records that match the selected filters.

- **Remove****Filters:** To view the complete list again, clear any applied filters.
