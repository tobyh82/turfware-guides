# How to use search

- **Locate****the****Search****Field:**The search field is typically positioned above the list. It can be found near the top-left of the section. (See figure below)

- **Enter****Search****Criteria:**Type the keyword or phrase you want to search for in the list(e.g., customer name, order number).

- **Initiate****Search:** Press the search icon or hit "Enter" to begin searching. The list will update to show only the records that match your search criteria.

- **Clear****Search:** To view all records again, clear the search field and initiate the search with an empty input.
