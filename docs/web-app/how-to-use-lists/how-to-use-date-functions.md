# How to use date functions

- **Locate****the****Date****Range****Picker:**The date range picker is usually situated near the top-right of the list. (See figure below)

- **Select****a****Date****Range:** Click on the date fields to open a calendar and choose your start and end dates. This allows you to filter the list by a specific time frame.

- **Apply****Date****Filter:**Once the dates are selected, the list will automatically refresh to show records that fall within the specified date range.

- **Clear****Date****Range:**To remove the date filter and see all records, clear the date range selection.
