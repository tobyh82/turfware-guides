# Concept of list, best practices

**Concept:** Alist is a way to organise and display data in a structured manner. It often includes columns for different attributes related to the data being managed. In the context of the figure below, a list is used to display confirmed orders, with columns such as Order Number, Delivery Date, Customer, Suburb, Payment Status, etc.

![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/22702/4350804/4._how_to_use_lists_-_Attachment_1.jpg)

**Best** **practices:**

- **Sorting****and****Filtering:**Utilise sorting and filtering options to find relevant information quickly.

- **Pagination:**Use pagination to manage large sets of data without overwhelming the user interface.
