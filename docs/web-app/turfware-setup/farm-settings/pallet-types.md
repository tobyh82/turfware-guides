# Pallet Types

**Pallet Types** are the pallet options available when harvesting an order (e.g. *Steel*). They feed the **harvesting pallet type** on an order and the pallet counts on the [Planner](../../dashboard/planner.md) and [Cut Sheet](../../dashboard/cut-sheet.md).

## Where to find it

Left-hand navigation → **Farm Settings → Pallet Types**.
