# Save and move status

- Save order: On the Order management page, click the green “**Save**” button at the top right corner to save the order detail updates.

- Move order to **ON****HOLD**: Click “**Hold**” from the dropdown list on the RHS of the“Save” button. (See Figure below).

- Move order to **LOST**: Click“**Lost**” from the dropdown list on the RHS of the “Save” button. (See Figure below).
