# Accepting Manual Payment

- On the Pending Order Management page, under “**PaymentStatus**”, click the green “**Received Payment**” button. (See figure A below)

- A box will appear allowing you to add the received payment method, date of received payment, payment reference and any notes/comments. (See figure B below).

![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/22702/4350819/5.3_orders_-_Attachment_6.jpg)  
Figure B

#### **How****does****it** **work**

- It allows the user to manually update the payment received. After accepting manual payment, the order status will be changed to **'Confirmed**.'

## Why and when to use this feature

- This feature is useful when payment is received through-automated methods, such as cash or bank transfer, and needs to be manually recorded in the system. Use this feature to update the order status to 'Confirmed' after verifying that the payment has been received and processed outside the usual Xero channel.
