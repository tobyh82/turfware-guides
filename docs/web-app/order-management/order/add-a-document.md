# Add a document

- To access Documents, click “Documents” on the navigation menu of the order.

- In the Documents tab, all uploaded documents are listed.

- To upload a new document, click the green "Upload" button in the top-right corner (see figure A below).

- A box will appear, allowing you to select the document type and choose a file from your local machine to upload. Then, click the green "Upload" button in the

bottom-right corner to complete the upload. (see figure B below)
