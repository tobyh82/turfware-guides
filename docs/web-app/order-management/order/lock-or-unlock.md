# Lock or unlock

**Lock****order:**Click the green “**LockOrder**” button which is next to the “**Send****Confirmation**” button. (See the Figure below)

![](https://static.helpjuice.com/helpjuice_production/uploads/upload/image/22702/4350826/5.3_orders_-_Attachment_13.jpg)

When an order is locked, those fields will be disabled on editing:

- Customer selection

- Segment

- Sales person

- Services Selection

- Pallet type

- Purchase order

- Market Channel

- Delivery location

- Freight Price

- Deliveries

- Installation (Layer, Lying)
