# Create “Pickup Only - Non-Farm location - Freight required” order

This operation allows you to create orders for pickup services at locations other than the farm. These locations could be additional dispatch points where your organisation offers pickup services to clients. Organisations can predefine these locations in the system, making them easily selectable as a pickup point during the order creation process.

### 

### **Key** **Points:**

**1.** **Quick****Selection****of****Non-Farm****Pickup** **Location**

By defining an on-farm pickup location, your organisation can quickly select this location during order creation. This ensures the order is correctly added to the freight and scheduling system, as it requires transportation from the farm location to the non-farm site.

## 

**2.** **Freight****and****Scheduling** **Integration**

Once a non-farm location is selected, the system automatically adds this order to the freight and scheduling queue. This ensures that the order is moved from the farm location to the designated farm pickup site.

## 

**3.** **Pricing****for** **Pickup**

Organisations can set a price for the non-farm pickup service. This can be either a free service or a standard delivery fee.

\*\* To create a freight-required pickup location, select 'Required' under 'Freight Required?' when creating or editing the pickup location (see figure below). Refer to Section 1.2.9 to learn more about the process of creating a pickup location.
