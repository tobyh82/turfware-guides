# Timeline

The Timeline is designed to provide detailed tracking of changes made to an order. This feature allows users to see what was modified, who made the change, and when it happened.

To access the Timeline page, click “Timeline” on the navigation menu of the order.

The timeline displays each change with the following details:

- **#****(ID)**: The unique identifier of the change record.

- **Description**: A brief summary of the change event(e.g., "Order Status Updated").

- **Audited**: Detailed information about what was changed, including old and new values.

- **Causer**: The name of the user who made the change, or "System" if it was an automated update.

- **Created****At** The timestamp showing when the change occurred.
