# What is on hold?

Orders that are marked as "**On Hold**" are set aside from active processes, meaning they don’t appear in Cutsheets or delivery runs. This status is intended for situations where the order needs to be paused due to outstanding issues, incomplete information, or waiting for customer confirmation.
