# The power of Hold, removed orders from Cutsheets and delivery runs

When an order is placed on hold, it is automatically removed from Cutsheets and delivery runs. This ensures that it doesn’t get processed or scheduled for delivery until the necessary conditions are met. Once the issue is resolved, the order can be moved back to an active status for normalprocessing.
