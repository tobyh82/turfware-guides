# How to manage quotes accepted status

- On the Left-Hand Side (LHS) navigation menu, under "Order Management," click "Quote Accepted" to access the Quote Accepted Management page.

- On the Quote Accepted Management page, you can filter quotes by accepted date using the date range filter. You can also use the search function to refine your results further.
