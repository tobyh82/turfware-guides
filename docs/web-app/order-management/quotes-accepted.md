# Quotes Accepted

**Quotes Accepted** lists every quote a customer has accepted. Getting here can be automatic (online acceptance) or manual (you convert it).

## Where to find it

Left-hand navigation → **Order Management → Quotes Accepted**.

## Online acceptance — what happens automatically

When a customer **accepts and pays** a quote through the online link, Turfware runs the workflow for you:

- updates the quote's status to **accepted**,
- generates the **invoice** in your accounting system (Sawfish),
- sends the customer an **acceptance email**, and
- creates the **order** (as Pending) so it can move into the workflow.

## Managing the list

Filter the list by **accepted date** using the date-range filter, and use search to narrow it further. You can also start a new quote from here with **Create Quote**.
